🟢 Google Sheets
Como conectar:
1. Crie uma planilha no Google Sheets
2. Vá em Arquivo → Publicar na web
3. Selecione a aba e formato CSV
4. Cole o link gerado abaixo
5. Para gravar dados, crie um Google Apps Script
SCRIPT APPS SCRIPT SUGERIDO:
function doPost(e) {
  var ss = SpreadsheetApp.openById('SEU_ID');
  var sheet = ss.getSheetByName('Agenda');
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.date, data.title,
    data.type, data.start,
    data.end, data.notes, data.color
  ]);
  return ContentService.createTextOutput('OK');
}
