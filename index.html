Como conectar:
1. Crie uma planilha no Google Sheets
2. Arquivo → Publicar na web
3. Selecione a aba e formato CSV
4. Cole o link abaixo
5. Para gravar dados, crie um Google Apps Script
Ver exemplo de Apps Script 
function doPost(e) {
  var ss = SpreadsheetApp.openById('SEU_ID');
  var sh = ss.getSheetByName('Agenda');
  var d = JSON.parse(e.postData.contents);
  sh.appendRow([d.date, d.title, d.type, d.start, d.end, d.notes, d.color]);
  return ContentService.createTextOutput('OK');
}
